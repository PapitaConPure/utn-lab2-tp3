Comenzando

Hay un par de cosas a considerar al abrir la aplicación.

Abriendo la aplicación por primera vez

Cuando abras la aplicación por primera vez, serás presentado con una ventana para ingresar el precio base de los alquileres de las propiedades ingresadas. Este es el precio que tendrán en cuenta los alquileres como base para sus demás factores.

A continuación, serás presentado con una ventana para ingresar tus datos de usuario y acceder al sistema.

Acceso al Sistema

La aplicación permite crear cuentas de diferentes tipos con un nombre de usuario y una contraseña.

Puedes ingresar a la aplicación con credenciales válidas de un usuario. Solo debes ingresar tu nombre y contraseña.

Creación de primer usuario

Si no existe ningún usuario, se creará un primer usuario como administrador al ingresar las credenciales solicitadas.

Tipos de usuarios

Hay 2 tipos de usuario con diferentes privilegios en el sistema:

Administrador Empleado
Puede hacer todo lo permitido por el sistema. Puede hacer altas y bajas de propiedades, además puede crear nuevos usuarios y anular reservas. Solo puede cambiar su contraseña, realizar reservas y respaldar archivos CSV solicitados de clientes y alquileres
